# Oribital-Strike-Cannon-DP

 **How to download?**
- 1.Download code as a .zip file 
- 2.Extract the .zip file
- 3.Open the folder called "Oribital-Strike-Cannon-DP-main"
- 4.Open the sub-folder called "Oribital-Strike-Cannon-DP-main"
- 5.Drag or copy and paste the folder called "OBC" into your world's "datapacks" folder
- 6.Enter the world in Minecraft, or if already in type "/reload" then you should be good to go!
